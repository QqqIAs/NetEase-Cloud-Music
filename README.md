### 基本功能
- 首页：轮播图、推荐歌单、推荐音乐&歌曲播放
- 搜索框：历史搜索记录、热搜榜&均具备搜索功能
- 搜索列表：搜索结果关键字高亮显示、歌曲播放
- 歌词展示：歌词实时滚动、点击具体歌词实现进度更新、评论区展示、相似歌曲播放
- 其他功能待完善中...
